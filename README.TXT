足りないもの：
これを動作させるには、ＮＪ氏の正規バイナリから、「rominfo.dat」「zipname.dat」が要ります。
ＮＪ氏の作成した「ＲＯＭ変換ツール」も要ります。
ＮＪ氏のページ：http://neocdz.hp.infoseek.co.jp/psp/index.html

ＮＪ氏の作成した「ＲＯＭ変換ツール」で「キャッシュファイル」を作成する必要もあります。
例：F:\PSP\GAME\cps2psp\cache\gigawing_cache.zip 4,735kB

「ＲＯＭ名は」「ＭＡＭＥ１．０７形式」に合わせる必要があります。
例：F:\PSP\GAME\cps2psp\roms\gigawing.zip 12,913kB （親セット）
例：F:\PSP\GAME\cps2psp\roms\gwingj.zip    1,302kB （日本版差分（追加）セット）

ちなみにＲＯＭもキャッシュもＺＩＰ形式のみ対応です。
（註：ＺＩＰはツールにより多少圧縮率が違うので、サイズはおおよそです。）
